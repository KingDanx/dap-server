Server Features

Welcome! Here are some key features of my Bun-powered server:

  • HTML Imports: Serve and bundle HTML files with automatic JavaScript and CSS processing using import <file> from "./file.html".
  • Static File Serving: Easily serve raw static files like images, stylesheets, and scripts from a public directory.
  • Dynamic Routing: Define routes with an intuitive routes object mapping paths to handlers or HTML content.
  • Fast and Lightweight: Powered by Bun’s ultra-fast JavaScript runtime and built-in bundler.
  • Simple API: Minimal and clean server setup with Bun.serve() — no extra dependencies needed.
  • Automatic MIME Types: Serves files with correct Content-Type headers based on their extensions.
  • Graceful Shutdown: Control your server lifecycle easily with the provided stop() method.

Try visiting / or /dashboard to see HTML import routes in action!
